Project Overview

We propose a visual hallucination detection scheme using in-context visual learning with pose guidance.

Abstract

Leveraging large-scale Text-to-Image (TTI) models has become common for generating exemplar or training datasets in generative fields. However, visual hallucinations with perceptually severe defects remain a concern, especially in non-photorealistic styles like cartoons. We propose a detection system using Vision-Language Models (VLM) for hallucination detection in NPR. Our system, pose-aware in-context visual learning (PA-ICVL), enhances VLM decision-making with pose guidance, significantly improving hallucination detection accuracy. By mitigating hallucinations, this research advances TTI modelsâ€™ applicability in non-photorealistic domains.
